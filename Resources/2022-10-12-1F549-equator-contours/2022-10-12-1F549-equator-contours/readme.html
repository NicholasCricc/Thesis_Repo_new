REFERENCES

Product: Contours, 30m resolution (SHP)

Data set name: JAXA AW3D30 Global Digital Surface Model

Vertical accuracy class: ~10 metres

Horizontal Coordinate Reference System (CRS): Web Mercator EPSG:3857

Vertical Coordinate Reference System (CRS): EGM96 height EPSG:5773
